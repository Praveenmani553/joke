# Joke Generator

A fun, animated joke generator built with HTML, CSS, and JavaScript.

## Features
- Colorful animated background
- Button with hover effects
- Random joke generation from a collection of 50+ jokes
- Responsive design that works on all devices

## How to Use
1. Click the "Give me a Joke!" button
2. Enjoy a random joke displayed in the joke box
3. Click again for another joke

## Deployment
This project can be easily deployed to Netlify or any static site hosting service.

## Technologies Used
- HTML5
- CSS3 (with animations and gradients)
- JavaScript (ES6)
